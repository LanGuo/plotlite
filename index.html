Welcome to Plotlite easy charts!!!!

Plotlite format guide:

  1. The first line should start with a chart type keyword, followed by your chart title. For chart type, we currently support scatter, bar plots.
  2. The second line starts with the xData keyword, followed by data to be plotted on the x-axis.
  3. The third line starts with the yData keyword, followed by data to be plotted on the y-axis.

An example Plotlite code:

scatter My awesome scatter plot xData 1,2,3,4,5,6,7,8 yData 2,4,6,8,10,12,14,16
